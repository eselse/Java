 ----------------------------------------------------------------------------------------------
| В классе Object определены перечисленные ниже методы, доступные в любом объекте.			   |
|----------------------------------------------------------------------------------------------|
| Object clone()	            	| Создает новый объект, аналогичный клонируемому объекту   |
|----------------------------------------------------------------------------------------------|
| boolean equals (Object объект)    | Определяет равнозначность объектов 					   |
|----------------------------------------------------------------------------------------------|
| void finalize()            		| Вызывается перед тем как неиспользуемый объект будет     |
|									| удален системой "сборки мусора"						   |
|----------------------------------------------------------------------------------------------|
| Class<?> getClass()     			| Определяет класс объекта во время выполнения			   |
|----------------------------------------------------------------------------------------------|
| int hashCode()       				| Возвращает хеш-код, связанный с вызывающим объектом      |
|----------------------------------------------------------------------------------------------|
| void notify()   					| Возобновляет работу потока, ожидающего уведомления от    |
| 									| вызывающего объекта 									   |
|----------------------------------------------------------------------------------------------|
| void notifyAll() 					| Возобновляет работу всех потоков, ожидающих уведомления  |
| 									| от вызывающего объекта								   |
|----------------------------------------------------------------------------------------------|
| String toString() 				| Возращает символьную строку, описывающую объект 		   |
|----------------------------------------------------------------------------------------------|
| void wait()						| Ожидает исполнения другого потока						   |
|----------------------------------------------------------------------------------------------|
| void wait(long миллисекунды)		|														   |
|----------------------------------------------------------------------------------------------|
| void wait(long миллисекунды,		|														   |
| 			int наносекнды)			|														   |
 ----------------------------------------------------------------------------------------------
